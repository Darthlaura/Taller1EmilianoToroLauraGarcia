Este taller tiene como desafio permitir a los usuarios registrar y gestionar actividades diarias con el objetivo de analizar su productividad y habitos. Metodos : 
Menu usuarios: permite a los usuarios registrar la actividad, modificar actividadm, eliminar actividad y cambiar contraseña. 
Menu analista: permite a los usiarios ver la actividad mas realizada, el usuario con mayor procrastinacion, y visualizacion de todas las actividades registradas 
En este taller se implementan contenidos vistos en clases como el uso del try y cath, uso de metodos con y sin parametros, arreglos nativos de java
